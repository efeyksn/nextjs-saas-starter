# Next.js SaaS Starter (Lite)

Tiny starter repo showing how I like to structure a SaaS-style Next.js app: app router, API routes and a simple pricing page.

This is intentionally minimal – it is here so hiring managers can quickly see how I write React/Next code without digging through big codebases.

## Tech

- Next.js 14 App Router
- React 18

## Structure

- `app/page.tsx` – marketing / pricing page
- `app/api/health/route.ts` – ultra-simple health check endpoint

## Getting started

```bash
npm install
npm run dev
```
